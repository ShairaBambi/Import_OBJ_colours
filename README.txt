OBJ Vertex Color Importer allows you to import your 3D scanned models into 3DS Max with Vertex Colour applied.

Browse - Choose your OBJ you want to import with vertex colours

Import - Imports chosen OBJ

NOTE: To see vertex colors on model, you must enable Vertex Colour within Object Properties.
View video for help. Or right click model and go down to Vertex Channel Display and enabling it,
making sure in the drop down menu you have chosen vertex color
